# DHG Core 4 — Platform Primer

This primer explains the role of DHG Core 4 within the larger Digital Harmony Group architecture.

---

## 1. Vision

DHG is building:

- A complete CME production and operations platform.
- Automated ingest → clip → caption → embed → RAG → publish pipelines.
- Studio OS integrations for vMix, Dante, NDI.
- An AI-assisted operator environment.
- Enterprise-ready search and proposal automation.

DHG Core 4 is the seed from which all these components grow.

---

## 2. Why This Repo Exists

- To establish architecture and folder structure.
- To define backend and frontend conventions.
- To ensure containerized local development.
- To provide a real, working, end-to-end baseline.

---

## 3. What Comes Next

Following this repo:

- MinIO integration
- Node-RED orchestration
- Worker pipelines
- ffmpeg + Whisper ingestion
- embeddings + vector search
- CME-specific tooling
- Studio control surfaces
